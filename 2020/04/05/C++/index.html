int main()

#int main()#

函数头 int main()
函数体 {}

int mian()被启动代码调用
启动代码是由编译器添加到程序中，是程序与操作系统之间的桥梁
int main()函数头描述的是main()和操作系统之间的接口
从关键字int可知，main()返回一个整数值
从空括号可知，main()函数不接受任何信息，任何参数

int main()
main()函数可以给调用它的函数返回一个整数值，且不从调用它的函数那里获得任何信息

int main(void)
在括号中使用关键字void明确地指出，函数不接受任何参数
在C++中，让括号空着与在括号中使用void等效
在C中，让括号空着意味着对是否接受参数表示沉默

void main()
在逻辑上一致，void返回类型意味着函数不返回任何值
该变体适用于很多系统，但它不是当前标准的一个强制选项，因此在有些系统上无法工作

如果编译器到达main()函数末尾时没有遇到返回语句，则认为main()函数以如下语句结尾

return 0
该隐含返回语句只适用于main()函数
